
# PredictiveModel3.0 Projekt

## Installation der Abhängigkeiten

Dieses Projekt verwendet eine `setup.py`-Datei, um alle erforderlichen Abhängigkeiten zu verwalten.

Folgen Sie diesen Schritten, um die Abhängigkeiten zu installieren:

1. Platzieren Sie die `setup.py`-Datei im Hauptverzeichnis Ihres Projekts.
2. Öffnen Sie eine Konsole und navigieren Sie zu dem Verzeichnis, in dem sich die `setup.py`-Datei befindet.
3. Führen Sie den Befehl `pip install .` aus, um die Abhängigkeiten zu installieren.

Dies sollte alle erforderlichen Pakete installieren.


Das Hauptprogram, in welchem die ML-Modelle trainiert werden heisst PredictiveModel x.x
In YahooNewData werden Daten bis zum heutigen Tag als Excel Datei gespeichert
In PredictionNewData x.x wird aus einem gespeicherten Modell und der Excel Datei eine Vorhersage gemacht
